WebNote

WebNote is a javascript library that allows for note-taking directly on web pages. This way, users don't need to download the web page and perform markup locally, but can perform the markup directly on the webpage!

WebNote Functionality

The various forms of note-taking include:

  • Highlight text in various colours
  • Cross out text
  • Add notes to text
  • Bookmark places within a webpage
  • Export the html of the page after note-taking
  • Import previous markup of the webpage
