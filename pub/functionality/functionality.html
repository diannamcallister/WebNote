Getting Started

There are 3 functions directly accessible when implementing WebNote:

1. nextToHighlight()

This function is the most commonly used function from the WebNote library. It is the function that creates the markup options bar, which will then be displayed on the webpage for the user to interact with.

Once the markup options bar appears, the user is able to select whatever action they would like to perform (highlight, strikethrough, create a note, add a bookmark, etc.). The action selected by the user will be processed through multiple helper functions within the WebNote library, the chosen markup will be displayed for the user, and the markup options bar will be removed from the webpage.

This function should be used in an event listener, whenever the desired text element is selected:
const textElement = document.getElementById("id_of_element");
textElement.addEventListener('click', nextToHighlight);

2. saveData()

This function should be used to save the html of a webpage, when a user has performed markup changes. This functionality would be best suited for websites who store their user's interaction with the page; this way, when a user leaves a webpage where they just performed markup, the webpage is able to save the markup performed by this user to a file, which can then be stored to the database of the website.

3. loadData()

This function should be used to load the html of the page, with previously created markup on it. This functionality would be best suited for websites who store their user's interaction with the page; this way, when a user is accessing a webpage where they previously performed markup, the webpage is able to load back in the markup performed by this user (as the webpage had previously saved the user's markup changes using the saveData() function from the WebNote library).
