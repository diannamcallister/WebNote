WebNote

About WebNote

Webnote is a javascript library that allows for note-taking directly on web pages. The various forms of note-taking include:

  • Highlight text in various colours
  • Cross out text
  • Bookmark different parts of the page for easy access

How to implement WebNote

Highlighting, striking out text, and adding a note are all performed by the same function in the WebNote library.

To implement this functionality:

  1. Figure out what text element you would like to add this functionality to on your webpage.
  2. Get the id of the text element you want to add this functionality to by doing:
    const textElement = document.getElementById("id_of_element");
  3. In your javascript file, add an event listener for when the chosen element is clicked, and have the function from the WebNote library called nextToHighlight be called:
    textElement.addEventListener('click', nextToHighlight);
  4. Now you're all set!
