Overview

All functionalities of WebNote are accessible through the markup bar displayed below:

User Interaction

When a user selects any text, the markup bar will appear. The user is then able to select any of the markup options to be performed on the selected text. The chosen markup option will be refelected on the webpage, and the markup bar will disappear.

Developer Interaction

To allow the user to access the markup bar, the developer must add an event listener to the text element where the markup functionality is desired. It can be added like:
const textElement = document.getElementById("id_of_element");
textElement.addEventListener('click', nextToHighlight);
nextToHighlight() is a function from the WebNote library which allows for the markup bar to be displayed!
